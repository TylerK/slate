Slate's UI Toolkit

Blocks

Blocks are the atomic level components that make up a complex user interface. Much like atoms, blocks are abstract in nature, don't provide a lot of use on their own, and can not be broken down into smaller components. Only when multiple blocks are combined do we start to see useful user interfaces take shape. Through the lense of web development, these are core HTML tags such as the <a>, <button>, or <input> and many other elements.

Blocks also encompass other globally available constructs such as typography, color pallets, tone and voice, animation styles, and other untangeable aspects that are used to craft complex user interfaces.
